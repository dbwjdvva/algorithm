# 🚀 Algorithm Log

> 📌 알고리즘 문제 해결을 위한 코드와 학습 내용을 관리합니다.

<br>

## 📂 폴더 구조

```plaintext
algorithm/
├── 백준/
│   ├── Bronze/
│   ├── Silver/
│   └── Gold/
├── SWEA/
└── README.md
```

- **백준/**: 백준 온라인 저지 문제 풀이 코드
  - **Bronze/**: 브론즈 레벨 문제 풀이
  - **Silver/**: 실버 레벨 문제 풀이
  - **Gold/**: 골드 레벨 문제 풀이
- **SWEA/**: 삼성 SW Expert Academy 문제 풀이 코드

<br>

## 🛠 사용 언어

- Java 11 이상
